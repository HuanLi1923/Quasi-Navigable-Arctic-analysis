# README

This repository contains the scripts used for data processing and visualization in the study.  

## Folder Structure

- **MATLAB/**  
  This folder includes MATLAB scripts for data processing.  
  These programs were used to preprocess and analyze the datasets before visualization.  

- **GMT/**  
  This folder includes Generic Mapping Tools (GMT) scripts for plotting.  
  These programs were used to generate the maps and figures presented in the manuscript.
